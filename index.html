Обо мне

Я — начинающий QA-инженер, совершенствующий свои навыки в области контроля качества и тестирования программного обеспечения на курсе онлайн школы - Нетология. В настоящее время активно изучаю автоматизированное тестирование на языке Java и другие современные технологии.

Моя цель — обеспечивать высокое качество выпускаемых продуктов, находить и документировать дефекты, тем самым внося непосредственный вклад в улучшение пользовательского опыта.

Помимо тестирования, я также имею опыт верстки на HTML и CSS, базовые знания SQL и SEO. В своей работе я активно применяю современный технологический стек: язык программирования Java для написания автотестов, инструменты разработки Visual Studio Code и IntelliJ IDEA, системы контроля версий Git и GitHub.

Я владею английским языком на уровне, достаточном для чтения технической литературы и документации. Нахожусь в постоянном поиске новых знаний и с интересом берусь за сложные задачи, чтобы расти как специалист. Готова к обучению и развитию в сфере QA и других сферах IT.
