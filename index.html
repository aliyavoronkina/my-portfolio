Обо мне

Я — начинающий QA-инженер, совершенствующий свои навыки в области контроля качества и тестирования программного обеспечения на курсе онлайн школы - Нетология. В настоящее время активно изучаю автоматизированное тестирование на языке Java и другие современные технологии.

Моя цель — обеспечивать высокое качество выпускаемых продуктов, находить и документировать дефекты.

Помимо тестирования, я также имею опыт верстки на HTML и CSS, базовые знания SQL и SEO. Недавно освоила автоматизированное тестирование на Java с использованием Selenium WebDriver, JUnit, TestNG. Умею тестировать REST API, работать с базами данных, использовать Docker для тестовых окружений. В своей работе я активно применяю современный технологический стек: язык программирования Java для написания автотестов, инструменты разработки Visual Studio Code и IntelliJ IDEA, системы контроля версий Git и GitHub.

Понимаю важность процессов Continuous Integration и Continuous Delivery в современной разработке. Изучаю инструменты CI/CD такие как Jenkins, GitHub Actions для автоматизации запуска тестов и обеспечения качества на каждом этапе разработки.

Я владею английским языком на уровне, достаточном для чтения технической литературы и документации. Нахожусь в постоянном поиске новых знаний и с интересом берусь за сложные задачи, чтобы расти как специалист. Готова к обучению и развитию в сфере QA и других сферах IT.

В работе активно использую современные AI-инструменты (ChatGPT, Claude, GitHub Copilot) для оптимизации процессов тестирования, генерации тест-кейсов и анализа кода. Это позволяет мне работать эффективнее и находить неочевидные сценарии тестирования.
